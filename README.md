# Blog front-end part

In this repository, the front-end part of the "Blog"

Using Material UI

## Present:

- User registration, login
- Writing and editing a post, with the possibility of uploading a img
- Deleting profile, posts, comments
- Commenting on posts
- Search by tags
- Global post search
- Sorting posts by date, popularity, "My posts"

Version 1.0 no adaptive mobile.

## Stack:

- ReactJS
- Redux Toolkit
- React Router
- React-hook-form
- Axios + Fetch
- React Hooks
- Material-UI
- CSS-Modules / SCSS
- React Content Loader (skeleton)
- React-paginate
- React-markdown
- MomentJS
- React Pagination
- Lodash.Debounce
